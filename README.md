# phishAvgSongLength
Scrape phishtracks.com, and find shows/sets with the highest average song length

Uses Python 2.7

Requred Packages:

Beautiful Soup 4 - http://www.crummy.com/software/BeautifulSoup/

pip install beautifulsoup4

lxml - http://lxml.de/

pip install lxml

If you dont want to download the files everytime, you can set the two download flags to False.
